TicTacToe

Rule: Each player has only 3 pieces. After placing 3, move by withdrawing a piece.

Place up to 3 pieces. Then select your piece to withdraw and place it on an empty cell. If you don't select, your last used piece will move automatically.

How to play

  • Manual turn selection using X / O buttons
  • Each player can place only 3 pieces
  • After 3 pieces, you must move (withdraw + place)
  • No selection means last used piece moves

Keyboard

Press R to reset board

Press N for new game (scores stay)
